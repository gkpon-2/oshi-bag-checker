# Oshi Bag Checker

推し活現場向けの持ち物チェックリスト生成アプリです。

現場タイプ・天気・気温・遠征有無に合わせて、必要そうな持ち物を自動でリストアップします。  
CLI版と、Streamlitを使ったWebアプリ版を実装しています。

## 概要

ライブ、舞台、接触イベント、オンラインイベントなど、現場ごとに必要な持ち物は少しずつ変わります。

Oshi Bag Checker は、現場タイプや当日の天気、気温、遠征の有無を選ぶだけで、持ち物チェックリストを作成できるPythonアプリです。

ブラウザ上でチェックリストを確認できるほか、txtファイルとしてダウンロードすることもできます。

## 主な機能

- 現場タイプに応じた持ち物リスト生成
  - ライブ
  - 舞台
  - 接触イベント
  - オンラインイベント
- 天気に応じた持ち物追加
  - 晴れ
  - 雨
- 気温に応じた持ち物追加
  - 普通
  - 暑い
  - 寒い
- 遠征用の持ち物追加
- 重複した持ち物の自動削除
- チェックリストの画面表示
- 画面上でのチェック操作
- txtファイルでのダウンロード
- CLI版でのチェックリスト生成
- CLI版でのtxt保存

## 開発メモ

このアプリは、Pythonの基礎学習と個人開発の練習を兼ねて作成しました。

最初にCLI版を作成し、その後Streamlitを使ってWebアプリ版を追加しています。

学習目的は以下です。

- Pythonの基本文法に慣れる
- 条件分岐とリスト操作を使う
- ファイル出力を実装する
- Pythonだけで簡単なWebアプリを作る
- GitHubでソース管理する
- Streamlit Cloudでアプリを公開する
