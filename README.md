# Oshi Bag Checker

推し活現場向けの持ち物チェックリスト生成アプリです。

現場タイプ・天気・気温・遠征有無に合わせて、必要そうな持ち物を自動でリストアップします。
CLI版と、Streamlitを使ったWebアプリ版を実装しています。

## 概要

ライブ、舞台、接触イベント、オンラインイベントなど、現場ごとに必要な持ち物は少しずつ変わります。

Oshi Bag Checker は、現場タイプや当日の天気、気温、遠征の有無を選ぶだけで、持ち物チェックリストを作成できるPythonアプリです。自由に持ち物を追加することもでき、ブラウザ上でチェックリストを確認したり、txtファイルとしてダウンロードしたりできます。

## 主な機能

- 現場タイプに応じた持ち物リスト生成（ライブ／舞台／接触イベント／オンラインイベント）
- 天気に応じた持ち物追加（晴れ／雨）
- 気温に応じた持ち物追加（普通／暑い／寒い）
- 遠征用の持ち物追加
- 重複した持ち物の自動削除
- チェックリストの画面表示・画面上でのチェック操作
- 自由に持ち物を追加（txtダウンロードにも反映）
- txtファイルでのダウンロード（Web版）／txt保存（CLI版）

## ファイル構成

```text
oshi-bag-checker/
├── main.py              # CLI版の起動ファイル（薄い入口）
├── streamlit_app.py     # Webアプリ版（画面まわり）
├── requirements.txt
├── README.md
├── .gitignore
├── output/              # CLI版の保存先（git管理外）
└── oshi_bag/            # ロジックをまとめたパッケージ
    ├── __init__.py
    ├── data.py          # 持ち物マスタ・ラベル定義
    ├── checklist.py     # チェックリスト生成・重複削除
    ├── output.py        # txt整形・保存
    ├── custom_items.py  # 自由追加アイテムの整理
    └── cli.py           # CLIの対話処理
```

## 使い方

### Webアプリ版（ローカル）

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

### CLI版

```bash
python main.py
```

## 持ち物の増やし方

持ち物や選択肢を増やしたいときは `oshi_bag/data.py` を編集します。ロジック側を触る必要はありません。

## 開発メモ

このアプリは、Pythonの基礎学習と個人開発の練習を兼ねて作成しました。最初にCLI版を作成し、その後Streamlitを使ってWebアプリ版を追加しています。

その後、長くなったスクリプトを責務ごとに `oshi_bag` パッケージへ分割し、各処理に日本語コメントを追加しました。あわせて、Web版でチェック操作をすると初期画面に戻ってしまう不具合を修正し、自由追加機能を実装しています。

学習目的:

- Pythonの基本文法に慣れる
- 条件分岐とリスト操作を使う
- ファイル出力を実装する
- Pythonだけで簡単なWebアプリを作る
- GitHubでソース管理する
- Streamlit Cloudでアプリを公開する
- モジュール分割で保守しやすい構成にする
